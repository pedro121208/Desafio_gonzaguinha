# 🎶 Projeto: O Que É, O Que É? - Gonzaguinha

> "Viver e não ter a vergonha de ser feliz..."

Este é um projeto de tributo digital dedicado a uma das obras mais importantes da Música Popular Brasileira: **"O Que É, O Que É?"**, composta e interpretada pelo mestre **Gonzaguinha**. O objetivo é criar uma experiência web que explore a letra, o ritmo e a filosofia por trás desta canção.

---

## 🚀 Sobre o Projeto

O site foi desenvolvido para ser uma plataforma interativa onde fãs e novos ouvintes podem mergulhar no universo de Luiz Gonzaga do Nascimento Júnior. Através de uma interface vibrante, o usuário é convidado a refletir sobre a pergunta central da música: *Afinal, o que é a vida?*

### ✨ Funcionalidades principais
* **Foco na letra** o foco principal fui ultilizar os versos e palavras da musica para divulgar a letra. E um site moderno com cores divertidas e limpas para a divulgação
---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando o "feijão com arroz" bem temperado do desenvolvimento web:

* **HTML5:** Semântica voltada para acessibilidade e SEO.
* **JavaScript (Vanilla):** Para manipulação do DOM e interatividade do player.
* **Bootstrap:** Usado para fazer o css e animação do site
